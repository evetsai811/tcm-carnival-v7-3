# 中醫嘉年華｜體質探索站 V7.3 正式版

本版重點：
- 手機版優先。
- 首頁使用直式森林大團圓封面。
- 每種動物 4 張圖：
  - 1：問卷前半段
  - 2：問卷後半段
  - 3：體質結果頁
  - 4：衛教建議頁
- 不使用四格 sheet 圖。
- 圖片以 contain 顯示，避免切掉耳朵、尾巴、翅膀。
- 平和體質同公式，需 ≥60 分才成立。

GitHub Pages：
1. 上傳 index.html、README.md、assets 資料夾。
2. Settings → Pages → Deploy from a branch。
3. Branch 選 main，Folder 選 /root，Save。
